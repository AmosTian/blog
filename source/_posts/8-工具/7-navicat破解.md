# 2. 破解

打开 Navicat 15 ---> 点击注册按钮 --->输入刚刚在破解工具中生成的注册码 ---> 点击激活 ---> 点击手动激活 ---> 将请求码复制到破解工具中 ---> 点击 Activation Code 一栏中的 Generate 按钮生成激活码 ---> 完成。
